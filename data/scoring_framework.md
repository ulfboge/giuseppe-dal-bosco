# Job-Fit Scoring Framework

_For evaluating job openings — Giuseppe Dal Bosco, June 2026_

Use this to quickly score any job posting before deciding whether to apply.
Score each criterion 1–5, then calculate the weighted total.

---

## Scoring Criteria

### 1. Forestry Technical Fit
Does the role require technical forestry skills present in the profile?

| Score | Meaning |
|---|---|
| 5 | Core technical forestry skills: silviculture, plantation management, natural forest ops, growth modelling, FSC, inventories |
| 4 | Strong overlap with 2–3 of the above; some adjacent skills required |
| 3 | Moderate overlap — forestry is one component among several |
| 2 | Forestry mentioned but not the technical core; agro/environment adjacent |
| 1 | Minimal forestry relevance |

---

### 2. Conservation / Restoration Fit
Does the role involve restoration, conservation, NbS, FLR, REDD+, or M&E of environmental outcomes?

| Score | Meaning |
|---|---|
| 5 | FLR, reforestation, NbS, REDD+, or restoration monitoring is the primary mandate |
| 4 | Conservation/restoration is a significant component (>50% of role) |
| 3 | Some conservation/restoration work, alongside other duties |
| 2 | Sustainability angle only; no direct restoration field work |
| 1 | Commercial/production focus only, no conservation dimension |

---

### 3. Seniority Fit
Does the role match senior/advisory/management level?

| Score | Meaning |
|---|---|
| 5 | Senior Specialist, Senior Advisor, Programme Manager, Director, Chief Forester or equivalent |
| 4 | Specialist or Manager (mid-senior); some leadership expected |
| 3 | Technical role with some autonomy; team lead or similar |
| 2 | Junior to mid-level; some gap in seniority expectations |
| 1 | Entry-level or clearly junior — mismatch |

---

### 4. Language / Region Fit
Does the role's language requirements and region match the profile?

| Score | Meaning |
|---|---|
| 5 | English, Spanish, or Italian required; region is Latin America, South Pacific, Africa, or Europe |
| 4 | English required; other multilingual advantage possible |
| 3 | English required; no specific regional language advantage |
| 2 | Primarily non-English working language (e.g., French, Portuguese as first language) |
| 1 | Role requires language not covered (e.g., Mandarin, French-only) |

> Portuguese: marked as Elementary in profile — be cautious about roles where Portuguese is the primary working language. [needs confirmation on actual working level]

---

### 5. Relocation Feasibility
How feasible is the location for relocation from Germany?

| Score | Meaning |
|---|---|
| 5 | Remote/flexible, international contract, based in Europe, or expat package offered |
| 4 | Based in a country with established expat infrastructure; relocation support common |
| 3 | Based in a country requiring visas/permits but feasible; no relocation support stated |
| 2 | Based in a difficult or high-complexity relocation country; support unclear |
| 1 | Fixed national hire only; residency required; no international recruitment |

---

### 6. Overall Application Priority
Summary judgement — weight all factors and gut-check organisational prestige/network value.

| Score | Meaning |
|---|---|
| 5 | Apply immediately — strong fit across all dimensions |
| 4 | Apply this week — good fit; minor gaps manageable in cover letter |
| 3 | Apply if time allows — moderate fit; worth a targeted application |
| 2 | Low priority — apply only if pipeline is thin |
| 1 | Do not apply — fundamental mismatch |

## Threshold Guide

| Total Score | Action |
|---|---|
| 25–30 | Apply immediately; prioritise cover letter customisation |
| 18–24 | Apply this week; standard tailoring sufficient |
| 12–17 | Apply if pipeline is thin; address gaps explicitly |
| < 12 | Skip unless strategic (network, referral, or bridging role) |
